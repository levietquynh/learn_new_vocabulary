* Assume on master branch, create Sign up branch *
1. git pull origin master
2. git checkout -b sign-up
3. git branch
4. <edit-change code>
5. git status
6. git add .
7. git status
8. git commit -m "Add file1, file 2, file 3"
9. git push origin sign-up
10. Create PR

* Rebase: Assume user is on branch want to rebase (sign-up)*
- Rebase not conflict 
1. Do something code on local
2. git status
3. git add .
4. git status
5. git commit -m "update file abc" 
6. git rebase master
7. git push -f origin sign-up
8. Create PR

- Rebase conflict
1. Do something code on local
2. git status
3. git add .
4. git status
5. git commit -m "update file abc"
6. git rebase master
7. Handle conflict (compare and accept current/incomming....)
8. git rebase --continue
9. git push -f origin sign-up
10. Create PR
